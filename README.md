# Task-2-Variables-Data-Types-Console-Input-Application
Interactive console-based calculator
## Description
This project demonstrates Java primitive data types, variable scopes,
console input using Scanner, arithmetic operations, type casting,
and formatted output.

## Features
- Interactive console calculator
- Type casting examples
- Input validation
- Clean and commented code

## Tools Used
- Java Development Kit (JDK) 25

## How to Run
1. Compile CalculatorApp.java
2. Run the program
3. Enter numbers as prompted

## Output
- Final output will be displayed using command prompt
